THAILAND SPATIAL DATA INFRASTRUCTURE (ThaiSDI)

โครงสร้างพื้นฐานข้อมูลภูมิสารสนเทศของประเทศไทย

เป็นระบบที่ถูกพัฒนาขึ้นตามมติคณะกรรมการภูมิสารสนเทศแห่งชาติปี ๒๕๕๕ มีวัตถุประสงค์เพื่อสนับสนุน และส่งเสริมการใช้ข้อมูลร่วมกันอย่างมีประสิทธิภาพระหว่างหน่วยงานต่างๆ ทั้งภาครัฐ ภาคเอกชนและสถาบันการศึกษา และองค์กรอิสระ ซึ่งจะช่วยลดปัญหาความซับซ้อนของข้อมูล และประหยัดงบประมาณในการดำเนินการเกี่ยวกับข้อมูล และประหยัดงบประมาณในการดำเนินการเกี่ยวกับข้อมูล นอกจากนี้ยังรวมถึงการสร้างระบบเครือข่ายเพื่อเชื่อโยงฐานข้อมูลของผู้ให้บริการข้อมูลต่างๆ ให้สามารถบริการข้อมูลที่ถูกต้อง ทันสมัย และตรงกับความต้องการของผู้ใช้ ทำให้ผู้ใช้ข้อมูลในระดับต่างๆ เข้าถึงข้อมูลได้ง่าย เพิ่มประสิทธิภาพในการใช้ข้อมูลร่วมกัน และสามารถสร้างและพัฒนาความร่วมมือทั้งในระดับท้องถิ่น ระดับประเทศ และระดับโลกระหว่างภาครัฐบาล ภาคเอกชน สถาบันการศึกษา และองค์กรอื่นๆ
